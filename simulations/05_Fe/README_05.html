How to run this simulation

  1. Preparation. Review makeparameterdict.R and makeMLfiles.R and run them in that order.
  2. Run response.surface.run.m in Matlab
  3. Postprocessing files are in postprocessing.

Parameters

  • Air temperature T
  • Total P concentration TP
  • Fe3+ concentration FE
  • DOC concentration DOC

Last three are controlled using runoff input to MyLake. This time there's no change in MyLake parameters.
